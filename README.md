# CODEASTRA 2026 - Online Coding Contest Platform

## Overview

CODEASTRA 2026 is a web-based online coding contest platform designed to host and manage competitive programming contests. The platform enables participants to register, join contests, solve programming problems, submit solutions, and view real-time leaderboards. It also provides administrators with tools to create and manage contests, problems, and users.

## Features

### Participant Features

* User Registration and Login
* Browse Upcoming, Active, and Completed Contests
* View Coding Problems
* Online Code Submission
* Real-Time Submission Status
* Contest Leaderboards
* Contest Dashboard

### Admin Features

* Create and Manage Contests
* Create and Manage Problems
* Manage Contest Schedules
* Configure Test Cases
* Monitor Submissions
* Manage Users

### Judge System

* Automated Code Compilation
* Execution Against Test Cases
* Time and Memory Limit Enforcement
* Result Generation:

  * Accepted (AC)
  * Wrong Answer (WA)
  * Time Limit Exceeded (TLE)
  * Runtime Error (RE)
  * Compilation Error (CE)

### Leaderboard System

* Real-Time Ranking Updates
* Score Aggregation
* Penalty-Based Tie Breaking
* Contest-Wise Rankings

1. Clone the repository

```bash

```

2. Open the project folder

```bash

```

3. Run the application locally using a browser or a local development server.
